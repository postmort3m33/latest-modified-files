# LatestModifiedFiles.ps1

A lightweight Windows GUI tool that recursively scans a folder and displays the most recently modified files — incase you just cant seem to remember that last file you were working on 10 files ago!

# Features:

GUI-based — no command-line arguments needed.
Recursive search — scans all subfolders automatically.
Sorted by last modified time — newest files appear at the top.
Horizontal + vertical scrolling for long paths.
